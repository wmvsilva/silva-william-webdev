The Movie Index (MI) is my final project for CS5610. It is hosted on Heroku and uses a Mongo mLab database. Using MI, users can log in to view and review their favorite movies! Users can maintain a profile with a description that shows all of the movies they have "liked".
Additionally, the MI allows users to sell movie products which other users can buy!
